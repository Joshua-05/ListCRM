## Настройка .env для сервера
Какие переменные понадобятся <br/>
- PORT - на каком порту запускать сервер, все остально зависит от того как в pgAdmin4 настроишь сервер <br/>
- DB_HOST 
- DB_PORT 
- DB_NAME 
- DB_USER 
- DB_PASSWORD
